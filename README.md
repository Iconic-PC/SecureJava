# Secure Java Client-Server System

## Description
Secure Java is a client-server application built using Java socket programming. The system enables secure communication between a client and server, demonstrating basic encryption and protected data transmission over a network.

## Technologies Used
- Java
- Socket Programming

## How to Compile
javac SecureServer.java  
javac SecureClient.java  

## How to Run
Start the server:
java SecureServer  

Then start the client:
java SecureClient
